\subsection*{Identified Bugs}
\begin{enumerate}
\item Address API explicitly refuses to ingest valid Pincode ints/strings yielding 400.
\item Cart implicitly approves bounds breaking quantity: 0 responses with Http 200.
\item Cart implicitly approves strictly subtractive limits via quantity: -5 breaking logic.
\item Review endpoints permit boundless positive scores (6) beyond ceiling thresholds.
\item Review endpoints permit boundless floor limits (0) below mathematical restraints.
\end{enumerate}
